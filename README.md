# Minicurso Aprendizado Federado Veicular SBRC 2026

Este repositório implementa a atividade prática desenvolvida para o Minicurso 2, entitulado Aprendizado Federado Veicular: da Teoria à Prática, apresentado no Simpósio Brasileiro de Redes de Computadores e Sistemas Distribuídos 2026.

```bibtex
@inproceedings{desouza2026aprendizado,

  author={de Souza, L. A. C. and others},
  booktitle={Minicursos do Simp{\'o}sio Brasileiro de Redes de Computadores e Sistemas Distribu{\'\i}dos (SBRC)},
  year={2026},
  organization={SBC}
}
```

# Preparação do Ambiente de Execução

Criar o ambiente de execução:

```bash
    source scripts/build/env.sh
```

Carregar o conjunto de dados:

```bash
    source scripts/build/datasets.sh
```
